# Agent Skills

A collection of agent skills for Claude Code and other coding agents.

## Install

```bash
npx skills add plimeor/agent-skills
```

Install a specific skill:

```bash
npx skills add plimeor/agent-skills --skill defuddle
```

## Skills

| Skill | Description |
|-------|-------------|
| [defuddle](skills/defuddle) | Fetch readable text content from any URL via defuddle.md |
